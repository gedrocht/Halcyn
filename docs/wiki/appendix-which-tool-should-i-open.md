# Appendix D: Which Tool Should I Open?

If you feel lost, use this page like a quick sorting hat.

## I want the main everything-is-open workflow

Use:

```powershell
.\scripts\launch-visualizer-workbench.ps1
```

## I want to launch things, open docs, or inspect the system from the browser

Use the browser Control Center.

## I want to shape source data into scenes and send them to the renderer

Use Visualizer Studio.

## I want to watch what every app is doing in one place

Use the Activity Monitor.

## I want to edit or explore browser-authored scene presets

Use Scene Studio.

## I want the exact field-by-field technical reference

Use the docs site and the generated code reference.

## I want the patient, guided explanation instead of the manual

Use this walkthrough wiki.
