# Halcyn Walkthrough Wiki

Welcome to the hosted version of Halcyn's slow, human, beginner-first learning
layer.

The main docs site at [the Halcyn Pages home](https://gedrocht.github.io/Halcyn/index.html)
is still the precise reference manual. This wiki is the calmer companion that
explains the same system in the order a new person usually needs.

## What this wiki is trying to do

This wiki is built like a guided course, not a field encyclopedia.

It tries to answer:

- what Halcyn is
- which tool is responsible for what
- how one change moves from an idea to pixels
- how to use the project without needing to understand every subsystem first
- how to grow from "I can run it" to "I can safely change it"

## If you only want one recommended path

1. Read [Start Here](01-start-here.md).
2. Read [Meet the Cast](02-meet-the-cast.md).
3. Open the workbench with `.\scripts\launch-visualizer-workbench.ps1`.
4. Read [Follow One Scene](03-follow-one-scene.md) while the tools are open.
5. Use [Your First Hour With Halcyn](appendix-first-hour.md) as a gentle lab.
6. Keep [Troubleshooting Without Panic](12-troubleshooting-without-panic.md)
   nearby for the first time something feels wrong.

## A calm promise

You do not need to understand the whole repo at once.

The real goal of this wiki is not to make Halcyn feel simple in a fake way. It
is to make Halcyn feel navigable, so you can always find the next understandable
piece.
