2D vertex fields

FieldMeaning
xThe horizontal position of the vertex in 2D space.
yThe vertical position of the vertex in 2D space.
rThe red color channel for that vertex, usually from 0.0 to 1.0.
gThe green color channel for that vertex, usually from 0.0 to 1.0.
bThe blue color channel for that vertex, usually from 0.0 to 1.0.
aThe alpha channel for that vertex, where 1.0 is fully opaque.

3D vertex and camera fields

FieldMeaning
x, y, zThe 3D position of the vertex.
r, g, bThe vertex color channels used by the shader.
camera.positionWhere the camera is located in world space.
camera.targetThe point the camera looks at.
camera.upThe direction treated as "up" for the camera.
fovYDegreesThe vertical field of view for the 3D projection.
nearPlaneThe near clipping plane distance.
farPlaneThe far clipping plane distance.

Runtime limit fields

The code now enforces maximum payload sizes, vertex counts, and index counts so mistakes become readable validation errors rather than mysterious crashes or frozen windows.

LimitPurpose
maxPayloadBytesCaps how much JSON one request can send.
maxVerticesCaps scene complexity at a beginner-friendly level.
maxIndicesCaps indexed mesh size for 3D submissions.
