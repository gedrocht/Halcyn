Fastest full check

.\scripts\run-all-quality-checks.ps1 -Configuration Debug

That script runs the Python quality gates for the browser Control Center and Visualizer Studio, prints the prerequisite report, verifies C++ formatting, and runs the native test suite when the machine has a supported toolchain.

Browser Control Center quality gates

.\scripts\lint-browser-control-center.ps1
.\scripts\typecheck-browser-control-center.ps1
.\scripts\measure-browser-control-center-coverage.ps1

Visualizer Studio quality gates

.\scripts\lint-visualizer-studio.ps1
.\scripts\typecheck-visualizer-studio.ps1
.\scripts\test-visualizer-studio.ps1
.\scripts\measure-visualizer-studio-coverage.ps1

Native Visualizer checks

.\scripts\verify-cpp-formatting.ps1
.\scripts\run-native-tests.ps1 -Configuration Debug
.\scripts\run-native-tests.ps1 -Configuration Release

Manual smoke path

  1. Launch .\scripts\launch-visualizer-workbench.ps1.
  2. Confirm the Visualizer window opens.
  3. In Visualizer Studio, preview and apply a preset scene.
  4. Switch to the Bar wall scene family and apply a bar-wall scene.
  5. Open the Activity Monitor and confirm that all participating apps are writing entries.
  6. Use the Control Center API Lab to hit /api/v1/health and /api/v1/examples/bar-wall.
