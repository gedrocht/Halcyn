What happened to this app?

The old shared data-source panel is no longer a separate supported desktop app. Its source-routing responsibilities are now part of Visualizer Studio.

What to use instead

.\scripts\launch-visualizer-studio.ps1

Visualizer Studio now owns source selection, scene-family selection, preview, validation, apply, live streaming, audio capture, and bar-wall scene generation in one place.
