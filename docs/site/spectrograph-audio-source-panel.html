What happened to this app?

The old standalone spectrograph audio-source helper is no longer the supported public workflow. Its job now lives inside Visualizer Studio under the Audio device source mode.

What to use instead

.\scripts\launch-visualizer-studio.ps1
  • Choose Audio device as the source mode.
  • Choose either Output sources or Input sources.
  • Start capture.
  • Switch between the preset scene family and the bar-wall scene family as needed.
