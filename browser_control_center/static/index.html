How to use this Control Center

A simple sequence for new users.

  1. Open Dashboard and confirm which tools are installed.
  2. Run Bootstrap for a human-readable prerequisite report.
  3. Use Build Debug once the toolchain is available.
  4. Start the app from the Build & Run section.
  5. Use API Lab and Smoke Checks to interact with the live API.
  6. Read the docs site for deeper architecture and code explanations.
